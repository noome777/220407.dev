1. 함수 선언

1) 선언적 함수

function 키워드에 함수명 작성하여 '선언'

2) 익명 함수

함수명을 작성하지 않고 변수에 '대입'

3) 자동 실행 함수

함수 정의와 동시에 즉시 실행되는 함수
한번만 호출 가능

4) 화살표 함수 (Arrow Function)

        ES6 에 추가된 함수

        [문법]
            - 매개 변수가 없을 때 : () => {...}
            - 매개 변수가 하나 일 때 : x => {...}
            - 매개 변수가 두 개 이상일 때 : (x,y) => {...}

            - 리턴
                - 처리할 라인이 여러개 라면, {}로 감싸서 처리
                (x,y) => {
                    ... 
                    return x+y;
                }
                -처리할 라인이 하나라면, {}와 return 생략 가능
                (x,y) => x+y;

2. 함수의 매개 변수

1) 매개 변수

함수 실행에 필요한 값을 외부에서 함수로 전달할 때, 매개변수를 통해서 매개값을 전달한다.
함수를 호출할 때, 매개변수의 갯수만큼 전달하는 것이 일반적이지만, 그렇지 않은 경우에도 에러가 발생하지 않는다.

2) arguments

초과된 매개값이나, 가변인자 전달 시 사용
