BOM(Browse Object Model)

window 객체

window 객체는 자바스크립트 최상위 객체가
브라우저 내 모든 요소들과 객체 담고 있음
BOM : location, navigator, history, screen

아이디 :

location 객체

브라우저의 주소표시줄 관련 객체

location.assign() location.replace() 두 메소드 모두 페이지 이동 (그러나 replace는 history에 기록하지 않음)

history 객체

브라우저의 방문 기록 관련 객체

navigator 객체

웹 페이지를 실행하고 있는 브라우저에 대한 정보를 가진 객체

screen 객체

운영체제의 화면(모니터)의 정보를 가진 객체
