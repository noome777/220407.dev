라디오 버튼


        라디오 버튼은 같은 name을 주어야 하나의 그룹으로 묶인다. 
        라디오 버튼은 하나의 값만 선택하게 할 때 사용한다. (다중선택x)
        기본 값으로 체크가 되어있게 하려면, checked 속성 부여
        제출(submit)시, check 된 input 태그의 value 가 전달된다.
        value 가 없으면, on 이라는 값으로 전달된다.






체크박스

        체크박스는, 다중선택을 할 때 사용한다. 
        체크박스는 선택 시 해당 태그의 value 가 전달한다.
        name 값은 똑같이 주면 된다. (어차피 전부 넘어간다.)
        기본적으로 체크될 요소들을 지정할 수 있다. (checked)

버튼 태그

       버튼은 기본적으로 버튼의 역할을 한다. (그냥 누를 수 있는)
       근데, 폼태그 내에서 버튼은 제출 역할을 한다. 
       버튼을 그냥 만들면, form 태그 내에서는
       type이 submit으로 지정된다.

       일반버튼(  || 은) 어디서든 존재할 수 있지만,
       서브밋버튼( + 서브밋버튼) 과 
       리셋버튼( + 리셋버튼)
       은 폼태그 안에서만 존재한다. 

       즉, form 태그 내에서 button 태그는 submit 을 기본값으로 가진다.

셀렉트 박스

            셀렉트 박스는,
            목록에서 값을 선택해서 서버로 전달한다.

            select - option 은
            option 태그의 content 영역에
            서버로 넘길 데이터를 넣어줄 수 있다.
            (value 속성을 이용해서 지정도 가능하다.)

            option 태그에 selected 속성을 추가하면
            기본적으로 선택이 되어있게 할 수 있다.







데이터 리스트

            목록에서 값을 선택하거나,
            직접 입력해서 서버로 데이터를 넘겨줄 수 있다.
